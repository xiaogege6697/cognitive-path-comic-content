# Changelog

## 0.2.0 - 2026-07-09

### 新功能

- 将恋爱转账纠纷示例扩展为完整交付链，覆盖 `EpisodeBrief`、`CognitivePath`、`PagePlan`、`RenderSpec`、`QAReport` 和 `PublishingPack`。
- 在 README 中补充“完整产物长什么样”和“出图落地方式”，明确逐页生成、逐页验收和后期叠字策略。
- 明确当前成熟模块为法律，其他行业作为可扩展主题使用通用模块模板。
- 将行为 eval 从3个扩展到10个，覆盖固定六页误用、三格过密、法律结论说满、中文错字放行、健康疗效承诺、绕过事实依据和局部失败恢复等场景。

### 调整

- 强化高风险行业边界：没有成熟行业模块或权威依据时，优先进入 `plan` / `prompts`，不直接生成最终图。
- 强化失败页局部修复原则，避免整组图文被单页失败拖住。

## 0.1.0 - 2026-07-04

### 新功能

- 建立视觉认知路径漫画内容生产系统。
- 支持动态5—8页、每页1—2格的分镜决策。
- 提供 `plan`、`prompts`、`generate`、`publish` 四种模式。
- 内置法律行业模块、通用行业模块模板和七道质量门禁。
- 提供恋爱转账纠纷示例及三张原创展示图。
