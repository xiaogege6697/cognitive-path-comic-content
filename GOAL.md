# 项目基因胶囊

## 目标

用视觉分镜控制用户认知路径，把复杂知识转化为愿意看、看得懂、愿意收藏并能行动的连续漫画图文。

## 成功标准

- 每期形成清晰的“停留—代入—事实—判断—反转—行动”认知路径。
- 动态规划 5—8 页，每页只承担一个认知动作并保持 1—2 格为主。
- 事实、适用范围、认知密度、人物连续性、中文排版和安全边界均通过质检。
- 输出可直接进入目标平台的生成或发布流程。

## 边界

- 认知路径优先于画风，不用漂亮插画掩盖逻辑和事实问题。
- 专业结论必须有事实依据；法律、健康等内容保留适用范围和安全声明。
- Core 只固定认知生产流程，行业、平台、角色和视觉实现可替换。

## 升级与验收

每轮只调整一个主要变量，用真实主题完整走一遍计划、提示词、逐页成图和七道质量门；至少一项有证据改善且核心质量不回退才接受。

## 稳定基线

当前发布基线为 `v0.1.0`，以 `SKILL.md`、示例成图和实际触发结果为准。
